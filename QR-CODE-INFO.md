## Your App URL
```
exp://192.168.100.116:8081
```

## How to Connect:

### Method 1: Manual URL Entry (Easiest!)
1. Download **Expo Go** app on your phone
2. Open Expo Go
3. Tap "Enter URL manually"
4. Type: `192.168.100.116:8081`
5. Tap "Connect"

### Method 2: Web Browser (No Phone Needed!)
1. Open your browser
2. Go to: http://localhost:8081
3. You'll see the Expo Dev Tools with a BIGGER, clearer QR code
4. Scan that QR code with Expo Go

### Method 3: Direct Web App
The app is running at: http://localhost:8081
- In the Expo Dev Tools, click "Run in web browser"
- Or press `w` in your terminal
